Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for the webpage. This element is extremely important for SEO.
  • The style element defines the CSS stlye associated with a webpage.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as character set, description, keywords, author and viewpoint.
  • Child Elements are elements nested iniside other elements.
  • The elements that holds all the children elements is called the Parent Element.
  • "Element" and "Tag" are commonly interchangable but an element refers to the entire element from opening tag to closing tag, whereas tag refers only to what appears inside the angle brackets.
  • The elements h1 throuhg h6 represent the level of heading a given text block represents.
  • The cp element represents a paragraph or a block of text.
  • The ul, ol, and li elements represent unorderded lists, ordered lists, and list items
  • Metadata is data about data, it always goes inside the head element.
  • The br element creates a line of empty space or a line berak between two blocks of text
  • the "a" element (stands for anchor) creates a link to the same webpage or other webpages.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Separation of Concern refers to separating code into diffrent files based on use.
  • DO NOT REPEAT YOURSELF (DRY) is a coding principle that encourages to reuse or share a pieces of code in order to reduce the number of lines of code.
  • The img selector assings property values to any image element in the markups.
  • The display property is asssigned eith the block value, which assings each image to start on a new line.
  • The rel attribute specifies the relationship between current documents and the linked document resource.
  • the href attribute specifies the location (URL) of the external source.
  • Declaration contains two important components: the CSS property we want to apply and the value of the property.
